The ultimate lizard tapping experience! A modern, feature-rich clicking game with stunning visuals, achievement system, and comprehensive SEO optimization.

## 🎮 Game Features

### Core Gameplay
- **Addictive Clicking**: Simple yet engaging one-click gameplay
- **Visual Feedback**: Satisfying animations and effects with each tap
- **Sound Effects**: Immersive audio experience with toggle option
- **Progress Saving**: Your score is automatically saved locally
- **Keyboard Support**: Play with Space or Enter keys

### Achievement System
- 🏆 **First 10!** - Unlock after tapping 10 lizards
- 🎉 **Century!** - Reach 100 lizard taps with celebration effects
- 👑 **Lizard King!** - Ultimate achievement at 1000 taps with epic celebration

### Visual Effects
- **Particle System**: Animated background particles for immersion
- **Floating Text**: Dynamic feedback with each tap
- **Glow Effects**: Beautiful lighting and shadow effects
- **Milestone Celebrations**: Special effects at score milestones
- **Rainbow Animations**: Epic celebrations for major achievements

### Utility Features
- 🔊 **Sound Toggle**: Enable/disable audio effects
- 🔄 **Game Reset**: Reset progress with confirmation
- 📤 **Score Sharing**: Share your achievements via native sharing or clipboard
- ℹ️ **Game Info**: Learn about features and controls

## 🚀 Technical Features

### PWA Capabilities
- **Installable**: Add to home screen on mobile and desktop
- **Offline Support**: Play without internet connection

### Performance Optimizations
- **Lazy Loading**: Optimized resource loading

- **Performance Monitoring**: Automatic particle management based on FPS
- **Responsive Design**: Perfect on all devices and screen sizes

### SEO Features
- **Comprehensive Meta Tags**: Title, description, keywords optimization
- **Open Graph**: Perfect social media sharing

## 🛠️ Technology Stack

- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Modern CSS with animations and responsive design
- **Audio**: Web Audio API with MP3 fallback
- **Storage**: LocalStorage for progress persistence
- **PWA**: Service Worker, Web App Manifest
- **SEO**: Structured data, meta tags, sitemap

